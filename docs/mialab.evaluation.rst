Evaluation (:mod:`mialab.evaluation` package)
=============================================

This package provides an easy way to evaluate the performance of your algorithms.

We provide an :class:`mialab.evaluation.evaluator.Evaluator` and two writers (:class:`mialab.evaluation.evaluator.CSVEvaluatorWriter`)
and (:class:`mialab.evaluation.evaluator.ConsoleEvaluatorWriter`), which can be used with a large amount of metrics
(see :mod:`mialab.evaluation.metric`).
